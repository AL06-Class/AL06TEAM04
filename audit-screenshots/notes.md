# design.md 화면 검토 메모

범위: 현재 Vite 메인 화면

증거:
- `01-main-desktop.png`: 1440x1000 데스크톱
- `02-main-mobile.png`: 390x920 모바일

확인 결과:
- 메인 레이아웃은 `design.md`의 좌측 카피, 우측 지도/추천 공고, 하단 3개 가치 카드 구조와 대체로 맞다.
- 색상 토큰, 카드 반경, 보라/라벤더 톤은 기준과 대체로 맞다.
- 활성 메뉴는 보라색 텍스트만 있고 하단 보라색 라인이 없다.
- 버튼처럼 보이는 요소가 `span`이라 실제 버튼/링크 동작과 접근성이 없다.
- 모바일 Header 메뉴는 접힘 처리되지 않고 세로로 펼쳐져 첫 화면 공간을 많이 차지한다.
- 데스크톱 히어로 카드 그림자가 기준보다 강하게 느껴진다.
- 화면 title이 `Docker Vite React`라 서비스 브랜딩과 맞지 않는다.

확인 명령:
- `bun run build`
- Playwright Edge 캡처
